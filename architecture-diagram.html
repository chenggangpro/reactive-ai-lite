Reactive AI Lite - Complete Architecture

Detailed Core Components & Design Principles

Fully Reactive Pipeline

  • • End-to-end non-blocking I/O leveraging Project Reactor
  • • Thread-safe operations with zero context-switching overhead
  • • Native WebClient streaming support for Server-Sent Events (SSE)
  • • Fluent Builder DSL for configuring executions

Composition & SPI Design

  • • Unified Provider SPI (Service Provider Interface) via Registry
  • • Composition over inheritance (LlmChatProviderDelegate)
  • • Drop-in support for OpenAI, Anthropic, DeepSeek, and Ollama
  • • Zero-friction autoconfiguration with Spring Boot

Robust Interceptor Chain

  • • Aspect-Oriented Out-of-band Interceptors
  • • Pluggable Logging, Metrics, and Validation steps
  • • Immutable Execution Contexts guarantee request integrity
  • • Chain-of-Responsibility design for pre/post processing
