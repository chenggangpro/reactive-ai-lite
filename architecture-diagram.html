reactive-ai-lite Architecture

Comprehensive architecture including Core Reactive Concepts, Delegation, Capabilities, and Client Structures.

Core Reactive Concepts

  • • Execution engine is built on Project Reactor
  • • Blocking responses mapped to Mono<GeneralResponse>
  • • Streaming mapped to Flux<StreamResponse>
  • • Fully non-blocking I/O throughout the pipeline

Delegate Pattern

  • • Isolates core framework from provider APIs
  • • Encapsulates HTTP request construction
  • • Normalizes JSON payloads to domain model
  • • Parses SSE stream fragments into structured deltas

Capabilities & Expanded Support

  • • Core resolves handlers via Capabilities enum (e.g. CHAT, EMBEDDING)
  • • Extend functionality by implementing LlmChatProviderDelegate
  • • Client packages structure properties, DTOs, and REST adapters
  • • Auto-registered dynamically via LlmProviderRegistry
